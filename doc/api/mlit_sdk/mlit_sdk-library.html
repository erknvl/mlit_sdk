mlit_sdk library

MLIT SDK for Flutter

A Dart/Flutter package that provides easy access to Japan's Ministry of Land, Infrastructure, Transport and Tourism (MLIT) APIs. This package includes functionality for retrieving real estate data, geographic information, facility data, and hazard information from MLIT's open data APIs.

The package features built-in caching for offline use, functional error handling using the Either pattern, and a clean service-based architecture.

Example usage:

// Initialize SDK
final mlitSdk = await createMLITSdk(
  MLITSdkConfig(
    apiKey: 'your_api_key_here',
    useCache: true,
    defaultCacheTtl: 3600, // 1 hour cache
  ),
);

// Get real estate transaction data
final result = await mlitSdk.transactionService.getTransactions(
  period: TransactionPeriod(year: 2023, quarter: 1),
  prefectureCode: PrefectureCode('13'), // Tokyo
);

// Handle result with functional error handling
result.fold(
  (failure) => print('Error: ${failure.message}'),
  (transactions) => print('Found ${transactions.length} transactions'),
);
